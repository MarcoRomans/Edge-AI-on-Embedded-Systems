# Edge-AI-on-Embedded-Systems
Progetto Edge AI su BeagleBone AI-64 dedicato all’esecuzione locale di Small Language Models Qwen2.5 quantizzati. Il lavoro valuta latenza, throughput, qualità delle risposte, stress CPU, isolamento tramite CPU pinning e rispetto di deadline real-time, confrontando modelli di dimensioni diverse in scenari embedded.
